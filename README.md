## Installation

1. **Clone the repository:**

    ```bash
    git clone https://github.com/yourusername/your-repo-name.git
    ```

2. **Navigate to the project directory:**

    ```bash
    cd your-repo-name
    ```

3. **Install the dependencies:**

    ```bash
    npm install
    ```

## Usage

To start the development server, run:

```bash
npm start
